Wallet

class Wallet(var activeKeyset: Keyset? = null, mintUrl: String, unit: EcashUnit)

A wallet handles all operations between a client and a mint. A single wallet is associated with a single mint and a single unit.

Parameters

activeKeyset

The Keyset that is currently active for the mint.

mintUrl

The URL of the mint.

unit

The underlying unit used with the ecash tokens for this wallet.

fun mint(amount: ULong, quote: MintQuoteResponse)

Request newly minted tokens from the mint. Note that this is done in two parts, this method being the second of the two. You must first ask the mint for a quote, pay the invoice it quoted, and then call this method. The mint returns a list of me.tb.cashuclient.types.BlindedSignatures, which the client unblinds and adds to its database.
